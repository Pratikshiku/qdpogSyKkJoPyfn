### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can be used to run)
- `IDEA`

### 3. Feature Description
The system is divided into (roles): students, enterprises, administrators.

**Student User Features:**
1. Login
2. Registration
3. User Management
4. Tripartite Agreement
5. Employment Policy
6. Employment Service Management

**Enterprise User Features:**
1. Login
2. User Information
3. Employment Policy
4. Teacher Module Features include Tripartite Agreement, Statistical Information, User Management

**Administrator Features:**
1. Login
2. User Management
3. College Information Management
4. Employment Module Features include Employment Management, Employment Service Management, Statistical Information, User Information
